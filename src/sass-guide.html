base/_functions.scss와 base/_mixins.scss의 항목을 각각 한 번씩 실제 스타일에 적용한 예시 페이지입니다.

함수 예시 (functions.scss)

rem()

px 값을 rem으로 변환합니다.

24px → rem 변환 텍스트

font-size: func.rem(24px);

em()

기준 폰트 크기를 기준으로 em 단위를 계산합니다.

부모(20px) 자식(24px 기준 em)

font-size: func.em(24px, 20px);

strip-unit()

단위를 제거해 순수 숫자로 변환합니다.

단위 제거 후 다시 px 조합

width: func.strip-unit(180px) * 1px;

lighten()

기본 색상을 더 밝게 만듭니다.

밝아진 primary 색상

background: func.lighten(vars.$primary, 12%);

darken()

기본 색상을 더 어둡게 만듭니다.

어두워진 primary 색상

background: func.darken(vars.$primary, 12%);

rgba-color()

색상에 투명도를 적용합니다.

primary 20% 투명도

background: func.rgba-color(vars.$primary, 0.2);

fluid()

clamp(min, preferred, max) 값을 빠르게 만듭니다.

화면 크기에 따라 너비가 바뀌는 박스

width: func.fluid(160px, 35vw, 360px);

fluid-font()

뷰포트에 따라 폰트 크기를 유동적으로 조절합니다.

모바일~데스크톱에서 폰트 크기 자동 조절

font-size: func.fluid-font(16px, 32px);
